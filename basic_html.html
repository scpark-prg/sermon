OpenAI API Key 등록 및 결제 방법

1. OpenAI 계정 로그인

  • https://platform.openai.com/ 에 접속하여 계정에 로그인하거나 회원가입합니다.

2. API 키 생성하기

  1. 오른쪽 상단의 프로필 아이콘을 클릭합니다.
  2. “View API Keys” 또는 “API 키 보기” 메뉴를 선택합니다.
  3. “Create new secret key” 버튼을 클릭하여 새 키를 생성합니다.
  4. 생성된 키를 복사하여 안전한 곳에 저장하세요. (다시 확인할 수 없습니다!)
    5. - [설교문 작성 프로그램의 API Key 입력 박스에 붙여넣고 "API Key 저장" 버튼을 눌러 저장하세요.]

3. 결제 수단 등록

  1. 프로필 아이콘을 클릭하고 “Billing” 또는 “결제”를 클릭합니다.
  2. Payment Methods 탭에서 신용카드 정보를 입력합니다.
  3. 청구지 주소까지 입력 후 저장합니다.

📌 주의사항

  • API 키는 외부에 노출되지 않도록 주의하세요.
  • 비용은 매월 사용량 기준으로 청구됩니다.
